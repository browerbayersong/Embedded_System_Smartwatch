# 整机电路原理与工作流程说明书

本系统主要由**主电源硬开关、电压转换、核心控制、外设传感器与显示**四个核心部分组成，实现了“一键开机/关机”及“低功耗全硬件自锁”的功能。

```
[3.7V锂电池] ──> [一键开关电路 (MOS管)] ──s1──> [AMS1117-3.3 稳压] ──VDD(3.3V)──> [STM32核心板 + OLED + MPU6050]
                         │                                                   │
                         └───────────────── 维持/关机信号 (PA0) ──────────────┘

```

---

## 一、 电路各部分原理简述

### 1. 一键开关电路（电源前端）

* **核心器件**：P-MOS (Q3) 作为总电源开关，N-MOS (Q4) 作为控制端，配合电容 C5 和电阻网络实现翻转。
* **原理**：利用电容 C5 暂存开机或关机瞬间的电荷状态。
* 关机时电容有电，按下按键注入 Q4 栅极使其导通，从而拉低 Q3 栅极实现上电自锁；
* 开机后电容电荷被抽空（变为 0V），再次按下按键时，它会将 Q4 栅极拉低，从而使 Q3 截止断电。


* **参数优化**：通过将 R2 减小至 $100\text{K}\Omega$，R8 维持 $1\text{M}\Omega$，使长按按键时 Q3 栅极电压逼近电池电压，**彻底消除了长按不松手时 MOS 管半导通发热、无法关机的隐患**。

### 2. LDO 电源稳压电路（中间级）

* **核心器件**：`AMS1117-3.3` 线性稳压芯片。
* **原理**：一键开关导通后输出的 `s1`（3.7V 锂电池电压）流入稳压芯片输入端。通过输入端（C3/C4）与输出端（C9/C10）的“大小电容组合”（100nF高频去耦 + 10uF低频滤波），将不稳定的电池电压平稳转化为标准的 **3.3V (`VDD`)** 供给全系统。

### 3. STM32 核心接口与 I2C 外设（后端主控）

* **核心器件**：STM32F103C8T6 核心板（Blue Pill）、MPU6050（加速度计）、OLED 屏幕。
* **原理**：
* STM32 通过排母引入 `VDD` 供电后启动工作。
* `PB6` 和 `PB7` 引脚配置为 I2C 复用开漏模式，利用外部上拉电阻 R5/R6（4.7K）维持总线高电平。
* MPU6050 和 OLED 共享该 I2C 总线，由单片机作为主控，通过不同的器件地址寻址，分时读取传感器姿态数据并刷新菜单显示。

## 二、 完整工作流程图解

整套系统从断电到运行的完整物理流程如下：

### 1. 待机状态（静态零功耗）

* 用户未按下按键时，Q4 栅极由于下拉电阻保持 0V 截止。
* Q3 栅极被 R2 上拉至 3.7V（$V_{GS} = 0V$），Q3 彻底关闭。
* 稳压芯片无输入，整个单片机系统没有任何电流消耗。

### 2. 触发开机时序

1. **用户按下按键 SW1**：电容 C5 的 3.7V 高电平注入 Q4 栅极，**Q4 导通**。
2. **MOS开关打开**：Q4 导通将 Q3 的栅极拉低至 0V（$V_{GS} = -3.7V$），**Q3 完全导通**。
3. **稳压输出**：电池电压穿过 Q3 变成 `s1` 送入 AMS1117，芯片输出纯净的 **3.3V (`VDD`)**。
4. **自锁维持**：单片机得电瞬间开始运行，`VDD` 经由 R7 持续向 Q4 栅极供电。此时用户松开按键，Q4 依然维持导通，**开机成功并自锁**。

### 3. 正常运行状态

* 单片机软件第一时间拉高 `PA0` 引脚，确保开关状态彻底锁定。
* 单片机通过 I2C 总线，以高频（如 50Hz）读取 MPU6050 姿态数据。
* 经过算法处理后，单片机再次通过 I2C 总线将画面数据发送给 OLED 屏幕，实现多级 UI 菜单的实时刷新。

### 4. 触发关机时序

1. **用户再次按下按键 SW1**：此时电容 C5 在开机期间已被放电至 0V，按下按键会将 Q4 的栅极瞬时强行拉低至 0V。
2. **解锁断电**：Q4 截止 $\rightarrow$ Q3 栅极被 R2 迅速拉高至 3.7V $\rightarrow$ **Q3 彻底截止**。
3. **整机断电**：`s1` 电压消失，LDO 停止输出，单片机及所有外设瞬间断电，电路重新进入“零功耗”待机状态。
